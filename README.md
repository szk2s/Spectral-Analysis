# Usage
### Type `edit main.m`, then run the sections one by one
You can convert output csv data into standard midi file with [Neuron](https://github.com/szk2s/Neuron)  
Hear the demo sounds [here](https://drive.google.com/drive/folders/1xU2hxmzMhu4SbaaS6ggDsQUYxpIQMYT8?usp=sharing)!!
